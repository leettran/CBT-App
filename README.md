# CBT-App
A basic CBT Thought Journal app.
<br><br>
#### A Thought Journal Entry consists of:
+ "Where were you?"
+ "Emotion or Feeling"
+ "Negative Automatic Thought"
+ "Evidence that supports that thought"
+ "Evidence that does not support that thought"
+ "Alternative thought"
+ "Emotion or Feeling"

<br>
